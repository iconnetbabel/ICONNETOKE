Pembayaran melalui Alfamart

1. Datang di gerai Alfamart terdekat

2. Info kasir untuk pembayaran DOKU ICONNet

3. Info kasir NO Pembayaran, Contoh : 111113000000001

4. Pembayaran : sesuai dengan jumlah pemesanan

5. Transaksi Selesai

6. Simpan Struk Pembayaran

Pembayaran melalui Indomaret

1. Datang ke gerai Indomaret terdekat

2. Info kasir untuk pembayaran Stroomnet

3. Info kasir kode indomaret 88267001 diikuti 8 digit Kode Pembayaran Contoh : 8826700110021136

4. Pembayaran sesuai dengan jumlah tagihan ditambah dengan biaya Rp 6.450,-

5. Transaksi Selesai

6. Simpan Struk Pembayarann

Pembayaran melalui Bank Mandiri

1. Melalui Mesin ATM Mandiri

2. Info Kode Biller 88008

3. Info No. Telepon User Id dan Password

4. Masukkan Jumlah Pembayaran

5. Klik Info Pada Saat PIN Internet Banking / Password

6. Masukkan OTP Kemudian Masukkan Jumlah Uang Bekerja

7. Transaksi Selesai

8. Simpan Struk Pembayaran
