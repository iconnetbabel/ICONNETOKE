Pembayaran melalui Alfamart

  • Datang di gerai Alfamart terdekat
  • kasir untuk pembayaran DOKU ICONNet
  • Info kasir NO Pembayaran, Contoh : 111113000000001
  • Pembayaran : sesuai dengan jumlah pemesanan
  • Transaksi Selesai
  • Simpan Struk Pembayaran

    • Pembayaran melalui Indomaret

  • Datang ke gerai Indomaret terdekat
  • Info kasir untuk pembayaran Stroomnet
  • Info kasir kode indomaret 88267001 diikuti 8 digit Kode Pembayaran Contoh : 8826700110021136
  • Pembayaran sesuai dengan jumlah tagihan ditambah dengan biaya Rp 6.450,-
  • Transaksi Selesai
  • Simpan Struk Pembayarann

    • Pembayaran melalui mandiri atm

  • Masukkan kartu ATM dan PIN
  • Pilih menu Bayar/Beli
  • Pilih Lainnya
  • Pilih Multi Payment
  • Masukkan kode perusahaan
  • Masukkan kode pelanggan
  • Transaksi selesai

    • Pembayaran melalui mandiri online

  • Buka aplikasi mandiri online di handphone atau PC
  • Masukkan user id dan pin di halaman log in
  • Pilih menu BAYAR -> BUAT PEMBAYARAN BARU
  • Pilih menu INTERNET -> ICONNET/89450
  • Pilih Rekening Sumber
  • Masukkan NO VA
  • Tekan Lanjut
  • Tekan KONFIRMASI
  • Masukkan MPIN Banking kemudian tekan OK
  • Transaksi Selesai

    • Pembayaran melalui mandiri cabang

  • Datang ke Cabang Bank Mandiri
  • Isi slip setoran atau pindah buku, Langkah Pengisian Aplikasi Setoran / Transfer
  • Isi Tanggal, Nama & Alamat Pembayar
  • Isi penerima dengan kode 89450
  • Isi No.Rekening dengan NO VA
  • Isi Jumlah Setoran & Terbilang
  • Isi Tujuan Transaksi. Misal: PEMBAYARAN XXXX
  • ntarkan slip setoran ke teller
  • Teller akan memproses & mencetak bukti pembayaran

    • Pembayaran melalui MCM / MIB

  • Buka aplikasi MCM / MIB
  • Masukkan company id, user id dan password
  • Pilih menu PEMBAYARAN -> PEMBAYARAN TAGIHAN
  • Pilih rekening debet
  • Pilih KATEGORI INSTITUSI (89450/ICONNET)
  • Masukkan NO VA
  • Tekan LANJUT -> KONFIRMASI
  • Approve transaksi dengan memasukkan pin dinamis token
  • Transaksi selesai

    • Pembayaran melalui BRI ATM

  • Masukkan kartu ATM dan PIN
  • Pilih menu Bayar/Beli
  • Pilih Lainnya
  • Pilih Multi Payment
  • Masukkan kode perusahaan
  • Masukkan kode pelanggan
  • Transaksi selesai

    • Pembayaran melalui Mobile Banking BRI

  • Buka aplikasi BRI Mobile
  • Pilih menu Mobile Banking BRI -> PEMBAYARAN -> BRIVA
  • Masukkan Nomor Virtual Account
  • Masukkan Jumlah Pembayaran
  • Masukkan PIN
  • Transaksi Selesai. Simpan notifikasi SMS sebagai bukti pembayaran

    • Pembayaran melalui Internet Banking BRI

  • Login pada alamat Internet Banking BRI (https://ib.bri.co.id/ib-bri/Login.html)
  • Pilih menu Pembayaran Tagihan > Pembayaran > BRIVA
  • Masukkan Nomor Virtual Account
  • Di halaman konfirmasi, pastikan detil pembayaran sudah sesuai seperti Nomor BRIVA, Nama Pelanggan dan Jumlah Pembayaran
  • Masukkan password dan mToken
  • Transaksi Selesai
  • Simpan Struk Pembayaran

  • Di halaman KONFIRMASI, pastikan detil pembayaran sudah sesuai seperti Nomor BNIVA, Nama Pelanggan dan Jumlah Pembayaran
  • Masukkan password dan mToken
